# Ad Spend Tracker - Quick Start

## No Installation Required!

Just run the PowerShell script - works on ANY Windows PC without admin access.

## How to Use

1. **Download the script:**
   - Get `ad_spend_tracker_bulk.ps1` from this repo

2. **Save to any folder:**
   - Example: `C:\Users\YourName\Documents\`

3. **Run the script:**
   - Right-click: `ad_spend_tracker_bulk.ps1`
   - Select: "Run with PowerShell"
   - Type: `go`
   - Press Enter

4. **Excel opens automatically and starts working!**

## Requirements

- Windows 10/11
- PowerShell (built into Windows)
- Microsoft Excel (optional but recommended)

## Troubleshooting

If script won't run:
```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser
```

Then run the script again.

## What It Does

Automates Amazon PPC campaign management tasks:

- Excel, Perpetua, Chrome, Teams integration
- Human-like behavior
- Smart randomization

## Stop the Script

Press `Ctrl+C` in the PowerShell window
